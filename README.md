
# Butterfly App Demo

This is a code project for Thinking Positive.


## Run Locally

Clone the project

```bash
  git clone https://github.com/SebasVelez34/butterfly-app
```

Go to the project directory

```bash
  cd butterfly-app
```
Use this version

```bash
  node v16.19.0 || nvm use
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start

```
